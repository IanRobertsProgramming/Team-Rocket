The Pokémon Encyclopedia

#Version 0.3.2 - 2021-04-13

About

The Pokémon Encyclopedia is a web application created for the access of information on generation I Pokémon. This application is to give Pokémon players access to a greater depth of information on Pokémon that is not provided by the in-game Pokédex.
