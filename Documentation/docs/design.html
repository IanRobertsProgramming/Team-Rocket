1.0 Introduction

The Pokémon Encyclopedia will provide fans of Pokémon and Pokémon players all around the world with a comprehensive database of relevant information on Pokémon which will allow them to make educated decisions on how they would like to play the game.

1.1 Purpose

This design document will describe the architecture, functions, design, software interfaces and features of the Pokédex Encyclopedia.

1.2 Scope

The intended audience of this document will be the developers of the Pokémon Encyclopedia website. This document will be easy to use and learn to help reduce the amount of time which is required to edit the website.

2.1.1 Updated Choice of Language
Front-end: HTML, CSS, Bootstrap
Back-end: Python, SQL, Javascript, PHP

Bootstrap:
Pokémon encyclopedia is a web-based application and in addition to HTML and CSS, Bootstrap is also used for the front-end development to offer an interactive application. The design team decided to add Bootstrap as one of the front-end languages, since it provides a great grid system and it cooperates well with HTML and CSS. Bootstrap is used to extend functionality of the Pokémon encyclopedia website, as it is a highly customizable language. If there is a demand for certain features, website developers can write code in Bootstrap and implement it on the web page with a third party add-on.

JavaScript:
In terms of the implementation of the Pokémon encyclopedia app, using JavaScript would be helpful for real-time applications, as JavaScript is an event-based language with asynchronous nature. Hence, it is a good choice for the Pokémon encyclopedia app, which requires constantly updated data.

PHP:
PHP is the top choice of language for the back-end implementation, since it is widely used to fetch data out of the database. This feature would provide users an ease of use experience, when they select their ideal Pokémon out of the comprehensive Pokémon encyclopedia database. Another advantage of using PHP is it allows direct output to HTML, therefore whenever there is a modification or deletion to the database, the webpage is updated correspondingly and spontaneously. PHP is free to download and supports a wide range of databases, overall, it is the best tool for the Pokémon encyclopedia application.
